# Neovim (LazyVim)

## Colorscheme

**jellybeans** (`nanotech/jellybeans.vim`)

```lua
return {
  { "nanotech/jellybeans.vim" },
  {
    "LazyVim/LazyVim",
    opts = {
      colorscheme = "jellybeans",
    },
  },
}
```
